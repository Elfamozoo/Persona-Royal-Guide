# 🎭 Le Navigateur du Voleur (Persona 5 Royal Guide & Tracker)

**Un compagnon interactif pour votre aventure dans Persona 5 Royal.**

Ce site est un guide complet et un tracker de progression 100% en français, conçu pour vous aider à maximiser votre temps et vos statistiques sociales lors de votre partie sur Nintendo Switch, PlayStation ou PC.

## ✨ Fonctionnalités

- 📅 **Journal du Destin :** Un calendrier interactif pour noter vos activités Matin, Après-midi et Soir.
- 📈 **Stat Tracker :** Suivez vos 5 statistiques sociales (Connaissance, Courage, Diligence, Charme, Gentillesse) avec les noms de niveaux officiels en français.
- 🧪 **Laboratoire de Fusion :** Un calculateur de fusions basé sur la logique de `chinhodado`, entièrement traduit en français.
- 💾 **Sauvegarde Locale :** Votre progression est enregistrée automatiquement dans votre navigateur.
- 📤 **Export/Import :** Exportez votre "save" sous forme de fichier JSON pour ne jamais perdre votre progression ou changer d'appareil.
- 📱 **Interface Optimisée :** Design inspiré de l'esthétique "Red & Black" de Persona 5, optimisé pour une utilisation sur mobile tout en jouant.

## 🚀 Installation & Lancement

Si vous souhaitez héberger ou modifier ce projet localement :

1. **Cloner le dépôt :**
   ```bash
   git clone git@github.com:Elfamozoo/Persona-Royal-Guide.git
   cd Persona-Royal-Guide
   ```

2. **Installer les dépendances :**
   ```bash
   npm install
   ```

3. **Lancer le serveur de développement :**
   ```bash
   npm run dev
   ```

4. **Accéder à l'application :**
   Ouvrez `http://localhost:5173` dans votre navigateur.

## 🛠️ Stack Technique

- **Framework :** React 18 (TypeScript)
- **Build Tool :** Vite
- **Styling :** Vanilla CSS (Persona 5 Aesthetic)
- **Persistence :** LocalStorage API
